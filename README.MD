# Dev Knowledge Hub

## Project Overview

The goal of this project is to create a simple web application that helps developers organize and store useful technical information related to different technologies they use in their daily work.

Developers often work with multiple tools, frameworks, and platforms. Each of them requires remembering various commands, documentation links, configuration notes, and troubleshooting steps. These pieces of information are often scattered across bookmarks, personal notes, chat messages, or documentation websites.

The purpose of this application is to provide a centralized place where developers can quickly store, manage, and access this knowledge.

The system should allow users to organize their information by technology and easily retrieve commands, links, and notes when needed.

## Main Goal of the System

The main goal of the system is to provide a simple and convenient way to:
- store useful commands
- save documentation links
- write notes related to specific technologies
- quickly access and copy stored information
- The system should help users reduce the time spent searching for commands and technical resources.

## Core Concept

The core concept of the application is organizing information around technologies.
Each technology acts as a container that can include:
- terminal commands
- useful links
- personal notes

## Features
Users can create and manage technologies they work with.

Each technology represents a specific tool, framework, or platform (for example: Docker, Git, Next.js, Kubernetes).

For each technology the user can:
- create a new technology entry
- view existing technologies
- edit technology details
- delete technologies

### Command Storage

The system allows users to store frequently used terminal commands related to a specific technology.

Each command may include:
- the command text
- an optional short description explaining what the command does

Users should be able to easily copy the command directly from the application.
### Link Management
Users can store useful links related to a technology.Users should be able to quickly open saved links directly from the application. When a user clicks a link, it should open in a new browser tab, allowing them to access the resource without leaving the application.
### Notes
The system should allow users to write notes related to a specific technology.
